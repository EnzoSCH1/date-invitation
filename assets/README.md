# Medias du projet

Place tes fichiers dans ce dossier avec ces noms :

- `photo.jpg` : la photo principale ronde de la premiere page
- `intro.gif` : le GIF de la premiere page
- `success.gif` : le GIF apres le clic sur Oui
- `final.gif` : le GIF du recap final

Tu peux utiliser `.jpg`, `.png`, `.webp` ou `.gif`, mais si tu changes l'extension, pense aussi a modifier le chemin correspondant dans `js/date-invitation.js`.
